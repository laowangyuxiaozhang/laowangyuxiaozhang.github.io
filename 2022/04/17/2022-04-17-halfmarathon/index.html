周六半马健康友谊跑步

周六半马健康友谊跑步（以下称周六跑）是为了教研室健康考虑，并虑及友谊强化的锻炼形式。

周六跑简介

周六跑始创于2022年4月，经过长足发展和积累，周六跑国际性、群众性、综合性、娱乐性和健康性不断提升，被自己教研室称为最具传播影响的锻炼形式之一。

周六跑项目

以下为周六跑项目：

  1. 男、女半程马拉松（21.095公里）

周六跑路线

以下为周六跑路线：
起点：东门 → 博留6公寓 → 研教楼后身 → 天桥 → 北门栅栏 → 大工桥下 → 生命学院后身 → 单向哈罗车道 → 绕经管学院 → 彩虹桥 → 地下通道北侧哈啰停放点 → 化环生楼 → 化环生楼后身 → 生物工程学院后身 → 西北门 → 西部田径场西侧台阶 → 绕大方楼 → 能动学院东侧 → 图书馆南侧 → 西部平价超市 → 蜂巢 → 西南篮球场 → 中心食堂南侧 → 田径健康中心里侧 → 沁园餐厅 → 西门 → 伯川西侧情人路 → 科技园餐厅 → 东门

单程全程7.5km
总计重复3次。

周六跑参加者要求

年龄

20周岁以上

身体要求

周六跑是一项大强度长距离的非竞技运动，也是一项高风险的非竞技项目。参加者应身体健康，有长期参加跑步锻炼和训练的基础，并根据自己的身体状况和锻炼能力，选择参加项目。

有以下疾病患者不宜参加周六跑：

  1. 先天性心脏病和风湿性心脏病患者；
  2. 高血压和脑血管疾病患者；
  3. 心肌炎和其它心脏病患者；
  4. 冠状动脉病患者和严重心律不齐者；
  5. 血糖过高或过少的糖尿病患者；
  6. 其他不适合运动的疾病患者。

跑步受伤了怎么办？

跑步受伤一般分急性损伤和慢性损伤。慢性损伤通常由长期错误跑步方式导致，也较难完全恢复，应以预防为主，并保持警惕。

急性损伤处理则分情况。擦伤时，如伤口创面较浅，通常只需涂红药水处理；如创面较大或有渗血，需用生理盐水和双氧水彻底清创后再涂红药水，渗血则需贴止血贴，严重者要及时就医。

扭伤和挫伤可以先冷敷（3天左右），消肿后再热敷，并辅以理疗进行修复，严重者需在医生指导下进行治疗。

另外，跑步时一定要时刻倾听身体的声音。特别是夏天，跑步中如出现头晕、恶心等中暑症状，或心率过快，一定及时停跑休息，必要时向周围人求助。
